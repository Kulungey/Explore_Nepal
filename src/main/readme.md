# Explore Nepal
Travel destination finder for tourists built with Java EE, JSP, MySQL and MVC architecture.

## How to Run
1. Import explore_nepal_schema.sql into XAMPP phpMyAdmin
2. Deploy on Tomcat 10
3. Open http://localhost:8080/ExploreNepal/login

## Tech Stack
- Java EE / Servlets
- JSP + CSS (Flexbox, no Bootstrap)
- MySQL (XAMPP)
- MVC Architecture
